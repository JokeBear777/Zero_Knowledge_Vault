## Zero Knowledge Vault

Zero Knowledge Vault는 “서버도 사용자의 비밀을 알 수 없어야 한다”는 Zero Knowledge 개념을 바탕으로 설계한 보안 금고 서비스입니다. 사용자는 마스터 비밀번호로 개인 금고를 열고 비밀번호, 계정 정보, 메모와 같은 민감한 정보를 저장할 수 있습니다.

이 프로젝트의 핵심 목표는 단순히 비밀 정보를 데이터베이스에 저장하는 것이 아니라, 서버가 평문 비밀번호나 금고 열쇠를 직접 알 수 없는 구조를 만드는 것입니다. 사용자가 입력한 마스터 비밀번호와 기기에서 관리되는 비밀 값을 기반으로 클라이언트에서 금고 열쇠를 복원하고, 실제 비밀 내용은 브라우저에서 암호화된 뒤 서버에는 암호문 형태로만 저장됩니다.

인증 과정에서는 SRP 기반 흐름을 참고하여 마스터 비밀번호가 서버로 직접 전송되지 않도록 설계했습니다. 또한 금고 내부 데이터는 VaultKey, itemKey, sharedItemKey 등 계층화된 키 구조를 통해 보호되며, 서버와 데이터베이스는 암호문과 최소한의 메타데이터만 저장합니다.

주요 기능으로는 마스터 비밀번호 기반 금고 잠금 해제, 개인 비밀 저장/조회/수정/삭제, 자동 잠금, 암호화된 인덱스 관리, 공유 금고 기반 비밀 공유 기능이 있습니다. 공유 금고에서는 공유 비밀을 별도의 sharedItemKey로 암호화하고, 각 참여자의 공개키로 sharedItemKey를 감싸 전달하는 방식으로 서버가 공유 비밀의 원문을 알 수 없도록 했습니다.

이 프로젝트를 통해 민감 정보 저장 서비스에서 필요한 클라이언트 사이드 암호화, 인증 흐름, 키 계층 설계, 권한 관리, 공유 비밀 관리 구조를 직접 설계하고 구현하는 것을 목표로 했습니다.
